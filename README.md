# Entry Point Github Repository for Harvard Dataverse Data Repositories
This is an entry point Github repo that links to several Harvard Dataverse repositories that Qijue Wang contributed to. Each repository contains: 
1. Experimental and/or simulation data
2. Programs for numerical simulation
3. Programs for data analytics

## Multi-Point Nanoindentation Method to Determine Mechanical Anisotropy in Nanofibrillar Thin Films
* Link: https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/LUEV5J

## Protein Secondary Structure in Spider Silk Nanofibrils
* Link: https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/GMEEYL

## Natural Spider Silk Nanofibrils Produced by Assembling Molecules or Disassembling Fibers
* Link: https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/P91YZ8
